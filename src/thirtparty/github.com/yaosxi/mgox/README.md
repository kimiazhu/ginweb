
## mgox

mgox is a friendly wapper of [mgo](http://gopkg.in/mgo.v2).

## Installation

    go get github.com/yaosxi/mgox

## Features

* Insert
* Remove
* Set/Inc/Push/Pull/Replace
* Find
* Get
